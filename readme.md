# Setup:
Rename default.env to .env and fill out all the fields

Set up your admin roles using commands:

 - /remove_admin_role
 - /add_admin_role
 - /add_admin_user
 - /remove_admin_user


Admins can either be a role, where it will apply to all users with that role, or a single user.
